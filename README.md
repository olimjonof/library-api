# 📚 Library RESTful API

A production-ready Spring Boot REST API for managing, searching, and renting books. Built using industry-standard practices, clean layered architecture, and Dockerized PostgreSQL.

---

## 🛠 Tech Stack

* **Java 17** (LTS)
* **Spring Boot 3.x**
* **Spring Data JPA** & **PostgreSQL**
* **Docker** (Containerized DB)
* **Maven**
